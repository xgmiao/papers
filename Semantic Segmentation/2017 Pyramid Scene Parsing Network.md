Pyramid Pooling Module

1.  将CNN最后输出的卷积特征图，分别池化为$1*1$、$2*2$、$3*3$、$6*6$的网格(类似SPPNet)
2.  对每个池化后不同尺度的feature map，采用$1*1$的卷积核来检测卷积核的个数
3.  将2中输出的特征图upsample到CNN输出大小
4.  将3中所有的特征图及CNN输出的特征图进行concat
